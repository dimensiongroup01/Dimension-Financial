# Dimension Financial Services - Next.js Site

Modern finance website built with:
- Next.js (App Router + SSR)
- React + TypeScript
- Tailwind CSS
- GSAP (scroll animations)
- Three.js (3D globe hero)
- Lenis (smooth scrolling)

## Run

```bash
npm install
npm run dev
```

## Notes
- Main SEO metadata is defined in `app/layout.tsx`.
- Financial services schema markup is injected in `app/page.tsx`.
- 3D scene and scroll effects are client-side with reduced-motion safeguards.
